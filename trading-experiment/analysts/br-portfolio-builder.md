## On VXUS's first two sessions: diversification confirmed, decoupling not

BW's report today is the clean answer to a question this desk could only estimate before: does VXUS actually reduce the book's chip-sector risk, or does it just add a second flag on the same map? Two live sessions now show the same pattern — VXUS falls *less* than NVDA on a chip-sector-driven selloff (-1.21% vs. NVDA's -2.80% today) but moves in the same direction, on the same catalyst, both days. The reason is structural, not incidental: VXUS's own top holdings include TSMC (3.98%), Samsung (2.19%), and SK Hynix (1.86%) — the same global chip-supply-chain names whose selloff (SK Hynix -11% overnight, SOX -15% from highs) is dragging NVDA and VTI down too. **Portfolio-construction conclusion: geographic diversification and sector diversification are two different jobs, and this book has now completed the first one and not the second.** VXUS was the correct next core dollar when it was proposed (7/15) because it closed a 0%-international gap that had gone unaddressed for three consecutive reports — that call stands. But it should not be mistaken for the fix to BW's Technology-concentration flag, which requires a satellite name with genuinely zero semiconductor/AI-compute linkage. OMCL remains the only candidate in the current pipeline that clears that bar cleanly.
